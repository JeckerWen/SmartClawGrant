# 课题申报书撰写技能

## 技能概述

本技能提供完整的课题申报书撰写工作流，包含四个阶段：
1. 课题申报网址获取（交互）
2. 研究现状调研与选题建议（自动）
3. 申报书内容要求确认（交互）
4. 申报书撰写与发送（自动）

## 阶段一：课题申报网址获取

**交互提示用户：**
```
请提供课题申报的网址或系统入口，例如：
- 国家自然科学基金委员会：https://isisn.nsfc.gov.cn/
- 国家社会科学基金官网：https://www.nopss.gov.cn/
- 某省科技计划项目申报系统：https://xxx.gov.cn/
```

获取网址后，使用 browser 工具访问并获取申报指南、课题方向、格式要求等信息。

## 阶段二：研究现状调研（自动）

### 2.1 调研任务

自动完成以下调研：
1. **国内研究现状**：搜索相关领域国内研究进展、主要研究团队、代表性成果
2. **国外研究现状**：搜索相关领域国际研究前沿、主要研究机构、最新进展
3. **研究热点与趋势**：分析领域内研究热点、发展趋势
4. **gap分析**：找出当前研究存在的问题和未解决的科学问题

### 2.2 调研工具

使用 web_search 和 web_fetch 进行调研，可根据学科领域调整搜索关键词。

### 2.3 输出调研报告

调研报告应包含：
- 研究背景与意义
- 国内外研究现状综述
- 研究热点与发展趋势
- 存在的主要问题（研究gap）
- 10个左右的选题建议（每个建议包含：题目、关键词、研究内容摘要、可行性分析）

### 2.4 发送到邮箱

将调研报告发送到用户邮箱（默认使用 TOOLS.md 中的邮箱配置）。

使用 email-sender 技能发送，邮件主题建议：`【调研报告】XXX领域研究现状调研`

## 阶段三：申报书内容要求确认

**交互提示用户：**
```
请提供以下信息：
1. 选定的选题题目
2. 申报书填写内容要求（如：研究目标、研究内容、创新点、预期成果等章节要求）
3. 有无特定的格式模板或字数限制
4. 其他特殊要求
```

## 阶段四：申报书撰写（自动）

### 4.1 申报书结构

根据课题类型，申报书通常包含：
- **立项依据**：研究背景、科学意义、应用前景
- **研究目标**：总体目标、具体目标
- **研究内容**：主要研究内容、关键科学问题
- **研究方案**：技术路线、研究方法、可行性分析
- **创新点**：理论创新、方法创新、应用创新
- **预期成果**：论文、专利、软件著作权等
- **研究团队**：团队成员、分工
- **经费预算**：各项经费合理性

### 4.2 撰写要点

- 语言严谨、专业、客观
- 突出创新性和可行性
- 格式规范、层次清晰
- 图表配合说明

### 4.3 发送到邮箱

将完整的申报书发送到用户邮箱，邮件主题建议：`【课题申报书】XXX课题申报书`

## 学科领域适配

根据不同学科领域，可能需要调整：
- 搜索关键词
- 报告结构侧重点
- 创新点的表述方式

常见学科参考：
- **医学/生物**：临床研究、基础研究、转化医学
- **计算机/AI**：算法创新、系统应用、产业落地
- **社科/经管**：政策建议、实证研究、案例分析
- **工程技术**：技术突破、工程应用、标准制定
